# Getting Started

## Installation

### NPM / Yarn

To install for use in CommonJS or ES module format:

```bash
npm i banshee-ui

yarn i banshee-ui
```

### Browser / UMD

To use in browser or other UMD builds include the following script:

`<script src="https://unpkg.com/banshee-ui@0.0.1/dist/BansheeUI.umd.min.js"></script>`
